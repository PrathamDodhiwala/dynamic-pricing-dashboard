# 📊 Dynamic Pricing Model Dashboard

A simple yet practical **Data Science project** that predicts the **optimal product price** based on demand, seasonality, and competitor pricing using Machine Learning.

Built with **Python, Scikit-learn, and Streamlit**, this project demonstrates an end-to-end workflow from data generation → modeling → interactive dashboard.


## 🚀 Features

* 📈 Predict optimal price based on:

  * Demand
  * Season (Peak / Off-season)
  * Competitor pricing
* 🤖 Machine Learning model using Linear Regression
* 📊 Interactive data visualizations
* 🎛 User-friendly dashboard using Streamlit
* 🧪 Synthetic dataset (500+ data points)


## 🛠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
* Streamlit

```bash

```

3. Run the app:

```bash
streamlit run app.py
```


## 💡 How It Works

* A synthetic dataset is generated with features:

  * Demand
  * Season
  * Competitor Price
* A **Linear Regression model** is trained to learn pricing patterns.
* Users input new values through the dashboard.
* The model predicts the **best selling price** instantly.


## 📊 Sample Insights

* Higher demand → higher price
* Peak season → price increase
* Competitor pricing influences final price
